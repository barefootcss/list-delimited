#List-delimited Object

The `list-delimited` object creates delimited inline lists.

	<ul class="o-list-delimited">
		<li></li>
		<li></li>
		<li></li>
	</ul>

## Installation

Install using Bower:

	$ bower --save bare-list-delimited

Import into project:

	@import "bower_components/bare-list-delimited/objects.list-delimited";
